# ROBOT FRONTIER

このプロジェクトは、CGプログラミングの課題制作物であり、以下のファイルを含んでいます。

## ファイル構成

- `index.html`: アプリケーションのメインHTMLファイルで、ゲームキャンバスを表示します。
- `Slide.html`: 新たに追加されたHTMLファイルで、デプロイ後にURL/Slideでアクセスできるように設定されています。
- `src/app.ts`: アプリケーションのエントリーポイントで、アプリケーションのロジックを実装します。
- `src/types/index.ts`: アプリケーションで使用される型定義を含むファイルです。
- `vite.config.ts`: Viteの設定ファイルで、ビルドや開発サーバーの設定を行います。
- `package.json`: npmの設定ファイルで、依存関係やスクリプトを定義します。
- `tsconfig.json`: TypeScriptの設定ファイルで、コンパイラオプションやコンパイル対象のファイルを指定します。

## インストール

プロジェクトをローカルにクローンした後、以下のコマンドを実行して依存関係をインストールしてください。

```
npm install
```

## 開発サーバーの起動

開発サーバーを起動するには、以下のコマンドを実行します。

```
npm run dev
```

## ビルド

プロジェクトをビルドするには、以下のコマンドを実行します。

```
npm run build
```
